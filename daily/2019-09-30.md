## jQuery

- 부모 요소 가져오기
    ```javascript
    //셀렉터로 잡히는 상위 요소 중 가장 근접한 하나
    $("selector").closet("selector")
    ```
    ```javascript
    //셀렉터로 잡히는 모든 상위 요소들
    $("selector").parents("selector")
    ```
- event 해당 요소
    ```javascript
    $("selector").on("click",function (e) {
        $(e.target)
    })
- class toogle
    ```javascript
    $("selector").toggleClass('className');
    ```
