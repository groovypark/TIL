# Python

### 순열(permutations)

```python
from itertools import permutations

people = [x for x in range(n)]
answer = list(permutations(people))
print(answer)    # [(0, 1, 2), (0, 2, 1), (1, 0, 2), (1, 2, 0), (2, 0, 1), (2, 1, 0)]
```
### divmod(a, b)

정수 나누기를 했을 때, 몫과 나머지로 구성된 한 쌍의 숫자를 돌려준다.
```python
    p,r=divmod(10,2)
    print(p, r)     # 5 0
```
